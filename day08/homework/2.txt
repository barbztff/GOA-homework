3. (ტექსტურ ფაილში) თქვენი სიტყვებით ახსენით რა არის input და output, მოიყვანეთ მაგალითები და დაურთეთ სურათები 

Input და Output განმარტება:

Input (შეყვანა) არის მონაცემები, რომლებიც შეჰყავთ პროგრამაში. ეს შეიძლება იყოს მომხმარებლის შეყვანილი ტექსტი, ფაილიდან წაკითხული ინფორმაცია ან სენსორებიდან მიღებული მონაცემები.

Output (გამოყვანა) არის მონაცემები, რომლებიც გამოდის პროგრამიდან. ეს შეიძლება იყოს ეკრანზე დაბეჭდილი ინფორმაცია, ფაილში შენახული მონაცემები ან სხვა სისტემაში გადაცემული შედეგები.

მაგალითი:

პროგრამა, რომელიც მომხმარებლის შეყვანილ რიცხვებს ამატებს და აჩვენებს შედეგს:

python
# Input - მომხმარებელს შეჰყავს ორი რიცხვი
num1 = input()
num2=input()


# Output - შედეგის გამოტანა
print("ჯამი არის:", num1 + num2)


ამ პროგრამაში:
- Input: მომხმარებელი შეყავს ორი რიცხვი
- Output: ეკრანზე გამოდის მათი ჯამი
