1. HTML(HyperText Markup Language) is the standart, used to create and structure content on the web. It provides the basic structure of web pages by defining the content, elements, and how they should be displayed by the web browser. HTML is extremely common and essential in the world of web development. In fact, it is the core building block for creating websites and web applications.

2. CSS (Cascading Style Sheets) is a styling, used to control the presentation, layout, and visual appearance of a web page written in HTML. While HTML provides the structure and content of a webpage, CSS defines how that content should look—such as colors, fonts, spacing, and positioning.

HTML defines the structure of the page (like headings, paragraphs, links, etc.), while CSS defines the style (colors, fonts, margins, etc.).
